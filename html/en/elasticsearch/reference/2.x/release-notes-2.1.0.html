2.1.0 Release Notesedit

Breaking changesedit

CAT API
  • Default _cat verbose to false #13180 (issue: #13156)
CRUD
  • Default detect_noop to true #11306 (issue: #11282)
Fielddata
  • Remove the experimental indices.fielddata.cache.expire #13173 (issue: #10781)
Index APIs
  • Add Force Merge API, deprecate Optimize API #13778
Internal
  • Forbid changing thread pool types #14367 (issues: #14294, #2509, #2858, #5152)
  • Remove usage of tuple as a method parameter #13135 (issue: #10787)
Java API
  • Deprecates defaultRescoreWindowSize #13995
  • Java api: remove execution from TermsQueryBuilder as it has no effect #12884
Mapping
  • Rename position_offset_gap to position_increment_gap #13056 (issue: #12562)
  • Default position_offset_gap to 100 #12544 (issue: #7268)
Nested Docs
  • If sorting by nested field then the nested_path should always be specified #13429 (issue: #13420)
Network
  • Move multicast discovery to a plugin #13027 (issues: #12993, #12999, #13019)
Packaging
  • Remove option to configure custom config file via CONF_FILE or -Des.default.conf #13772 (issues: #12712, #12954, #13715, #5329)
Plugins
  • Use name from plugin descriptor file to determine plugin name #12775 (issue: #12715)
Search
  • Limit the size of the result window to a dynamic property #13188 (issue: #9311)
Settings
  • Remove environment from transport client #13383 (issue: #13155)
  • Do not permit multiple settings files #13043 (issue: #13042)
Stats
  • The queue_size value should be shown as an integer. #13063 (issue: #10404)

Deprecationsedit

Java API
  • Deprecate the count api in favour of search with size 0 #14165 (issue: #13928)
Parent/Child
  • Deprecate score_type option in favour of the score_mode option #13478
Query DSL
  • Deprecate NotQueryBuilder #14203 (issue: #13761)
  • Query DSL: deprecate _name and boost in short variants of queries #12966 (issue: #11744)
Search
  • Deprecate _search/exists in favour of regular _search with size 0 and terminate_after 1 #13910 (issue: #13682)
  • Deprecate the scan search type. #12994 (issue: #12983)

New featuresedit

Aggregations
  • Adds geo_centroid metric aggregator #13846 (issue: #13621)
  • Add percentiles_bucket pipeline aggregation #13186
  • Add stats_bucket / extended_stats_bucket pipeline aggs #13128
Analysis
  • Lithuanian analysis #13244
